# AccessGate AI - Project Summary

## One-Liner
*"Full-stack location attestation system with AI-powered policy generation, built with Next.js, FastAPI, and PostgreSQL."*

- **Frontend**: Next.js 14, React, TypeScript, Tailwind CSS, next-themes
- **Backend**: FastAPI, Python, Pydantic, SQLAlchemy, Alembic
- **Database**: PostgreSQL (Docker)
- **AI**: OpenAI GPT (with Mock fallback)
- **Infrastructure**: Docker Compose
